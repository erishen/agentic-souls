# Memory: 投资分析数据准确性问题

## 问题 ID
`MEM-003`

## 发现时间
2026-05-17

## 问题描述

### 现象
在投资分析中引用了多个过时或不准确的数据：

1. **银行3年定期利率**：引用为 ~2.0%，实际已降至 **1.25%**（2026年5月1日新规）
2. **基金A类/C类差异**：引用A类5年年化5.37%评价用户持有的C类基金，实际C类年化仅2.12%
3. **霍尔木兹海峡通航量**：首次引用下降30%，实际下降95%

### 影响
1. 投资建议基于错误数据，可能导致错误决策
2. 降低分析报告的可信度
3. A类/C类差异导致对基金收益的错误判断

### 根因分析
1. 银行利率数据未及时更新，仍使用2023-2024年的旧数据
2. 未区分基金A类和C类的费率结构和收益差异
3. 地缘政治信息首次搜索不够深入，后续搜索才获取准确数据

## 解决方案

### 数据验证规则

#### 银行存款利率（2026年5月最新）
| 银行类型 | 1年 | 2年 | 3年 | 5年 |
|----------|-----|-----|-----|-----|
| 国有六大行 | 0.95% | 1.05% | 1.25% | 1.30% |
| 股份制(中息) | 1.15% | 1.20% | 1.30% | 1.35% |
| 股份制(高息) | 1.30% | 1.45% | 1.85% | 1.85% |
| 城商行/农商行 | 1.35-1.60% | 1.40-1.65% | 1.65-1.95% | 1.70-1.95% |
| 大额存单(20万起) | - | - | 1.55-1.95% | - |

#### 基金A类/C类差异
| 对比项 | A类 | C类 |
|--------|-----|-----|
| 申购费 | 有(0.1-1.5%) | 无 |
| 销售服务费 | 无 | 有(0.1-0.6%/年) |
| 赎回费 | 持有越久越低 | 持有7-30天免 |
| 适合 | 长期持有(>1年) | 短期持有(<1年) |
| 长期收益 | 更高 | 更低 |

#### 分析前必须确认
1. 用户持有的是A类还是C类
2. 银行利率是否为最新（每月5日可能更新）
3. 地缘政治信息需多次搜索交叉验证

## 验证方法

1. 引用银行利率时，标注数据来源和日期
2. 分析基金时，明确区分A类/C类
3. 重大地缘事件数据，至少搜索2次交叉验证

## 教训总结

1. **银行利率变化快** - 2026年5月新规后利率大幅下调，不能使用旧数据
2. **A类≠C类** - 同一基金A类C类收益差异显著，必须区分
3. **交叉验证** - 重大信息需要多次搜索确认，不能只看一次结果
4. **标注数据日期** - 引用数据时必须标注"截至X月X日"
